# Endpoints

**We are moving stuff around currently following the rebrand, guidance will follow soon!**

You can connect to the API on the following URLs:

| URL                           |  Release   | Description                        |
| ----------------------------- | :--------: | ---------------------------------- |
| `https://stoat.chat/api`      | Production | Primary API endpoint               |
| `https://api.revolt.chat`     | Production | Former Primary API endpoint        |
| `https://app.revolt.chat/api` | Production | Former API endpoint for old client |
| `https://revolt.chat/api`     |  Staging   | Former API endpoint for new client |

You can connect to the events server on the following URLs:

| URL                            |  Release   | Description                           |
| ------------------------------ | :--------: | ------------------------------------- |
| `wss://stoat.chat/events`      | Production | Primary events endpoint               |
| `wss://ws.revolt.chat`         | Production | Former Primary events endpoint        |
| `wss://app.revolt.chat/events` | Production | Former Events endpoint for old client |
| `wss://revolt.chat/events`     |  Staging   | Former Events endpoint for new client |
